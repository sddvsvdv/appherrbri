Cómo funciona

Haz una foto del residuo

Intenta fotografiar un solo objeto, centrado y con buena luz.

La app muestra si corresponde a papel y cartón, plásticos, vidrio, metales, cartón para bebidas, orgánicos, resto, punto limpio, textil, voluminosos u otro destino configurado.

Resultado rápido: EcoFot IA está pensada para abrir la app, analizar el residuo y recibir una respuesta clara, sin pantallas complicadas ni conversaciones largas.

Una app para saber dónde botar cada residuo

Muchas dudas de reciclaje aparecen con objetos cotidianos: envases, botellas, frascos, latas, cartón para bebidas, restos orgánicos, objetos rotos o residuos que no encajan claramente en una categoría.

EcoFot IA está pensada para ayudarte en esas situaciones. No se limita a decir “plástico = reciclaje”. El objetivo es orientar el resultado según el tipo de residuo y las reglas configuradas para el sistema de reciclaje de Chile.

Ejemplo: una botella de vidrio no es lo mismo que un vaso de cristal roto. EcoFot IA intenta diferenciar el objeto para dar una respuesta más correcta.

Destinos habituales en Chile

En Chile, EcoFot IA puede orientar el residuo hacia destinos como:

  • Papel y cartón: papel y cartón limpios.
  • Plásticos: envases plásticos limpios.
  • Vidrio: botellas y frascos de vidrio.
  • Metales: envases metálicos vacíos.
  • Cartón para bebidas: envases multicapa de bebida.
  • Orgánicos: restos de comida y residuos vegetales.
  • Resto: residuos que no encajan en otros destinos habituales.
  • Punto limpio / residuos especiales: residuos especiales, electrónicos, peligrosos o que requieren recogida específica.
  • Textil: ropa, calzado y textiles cuando haya recogida específica.
  • Voluminosos: muebles, colchones y objetos grandes.
  • No parece residuo: cuando la foto no muestra claramente un objeto que se quiera botar.

Preguntas frecuentes

¿EcoFot IA es una app para saber dónde botar cada residuo?

Sí. La app está pensada para ayudarte a decidir el destino de un residuo usando una foto.

¿Funciona como un chatbot?

No. EcoFot IA está diseñada para mostrar un resultado directo: destino, color y una explicación breve.

¿Necesita Internet?

Sí. EcoFot IA necesita conexión a Internet para analizar imágenes mediante servicios externos.

¿La inteligencia artificial decide directamente el destino?

La inteligencia artificial ayuda a identificar el objeto. Después, la app aplica reglas internas para mostrar el destino configurado.

¿La IA puede saber siempre de qué material está hecho un residuo?

No siempre. La IA interpreta la imagen de forma aproximada, usando patrones visuales, información habitual sobre objetos y reglas de clasificación. No puede confirmar al 100% el material real si no se ve claramente.

¿Sirve para Chile?

Sí. Esta versión está pensada para orientar el reciclaje según el sistema de residuos configurado para Chile.

¿Puede equivocarse?

Sí. La inteligencia artificial puede cometer errores, especialmente si la foto es confusa, hay varios objetos o el residuo no se ve bien.

¿Qué hago con residuos peligrosos o especiales?

Si el residuo puede ser peligroso, electrónico, sanitario, químico o voluminoso, consulta siempre las indicaciones oficiales de tu municipio o punto limpio.

¿Es gratuita?

La app puede incluir usos gratuitos iniciales, anuncios recompensados o suscripción. Consulta las condiciones actualizadas en Google Play.
