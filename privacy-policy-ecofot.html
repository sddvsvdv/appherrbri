Política de Privacidad de EcoFot

Última actualización: 01/05/2026

1. Información general

EcoFot es una aplicación desarrollada por Aplicaciones HerrBri cuyo objetivo es ayudar a identificar el destino adecuado de residuos mediante el análisis de imágenes o texto.

2. Datos personales

EcoFot no recopila directamente datos personales identificables como nombre, dirección o teléfono.

3. Uso de imágenes y texto

La aplicación permite enviar imágenes o texto para su análisis.

Cuando el usuario realiza un análisis:

  • La imagen o texto se envía a un servidor seguro gestionado por el desarrollador
  • El servidor procesa la información utilizando servicios de inteligencia artificial
  • Se devuelve un resultado al usuario

Las imágenes y textos:

  • No se almacenan de forma permanente
  • Se utilizan exclusivamente para generar el resultado solicitado

4. Servicios de terceros

Procesamiento de inteligencia artificial

El análisis puede implicar el uso de servicios externos de inteligencia artificial, que pueden procesar temporalmente la información enviada.

Publicidad (Google AdMob)

La aplicación muestra anuncios a través de Google AdMob.

AdMob puede recopilar:

  • Identificadores de dispositivo
  • Datos de uso
  • Ubicación aproximada

Más información: Google Privacy Policy

Compras dentro de la aplicación

Las compras y suscripciones son gestionadas por Google Play. El desarrollador no tiene acceso a los datos de pago.

5. Uso de Internet

EcoFot requiere conexión a Internet para funcionar, ya que el análisis se realiza mediante servicios externos.

6. Contacto y envío de información por el usuario

La aplicación puede ofrecer la opción de enviar un correo electrónico para reportar resultados incorrectos o proporcionar feedback.

En ese caso:

  • El usuario envía voluntariamente la información
  • El contenido del correo puede incluir imágenes o resultados
  • El desarrollador podrá utilizar dicha información para mejorar la aplicación
  • El desarrollador podrá responder al usuario a través del mismo canal

7. Base legal (GDPR)

El tratamiento de datos se basa en:

  • Consentimiento del usuario para el análisis de imágenes y uso de servicios
  • Interés legítimo para el funcionamiento técnico de la aplicación

8. Transferencias internacionales

Algunos servicios pueden procesar datos fuera del Espacio Económico Europeo, con las garantías legales correspondientes.

9. Derechos del usuario

El usuario puede:

  • Retirar su consentimiento en cualquier momento
  • Solicitar información sobre el tratamiento de datos

10. Cambios en esta política

Esta política puede actualizarse en cualquier momento. Se recomienda revisarla periódicamente.
